# Task 2: Bus Ticket Booking System Prototype

This task contains a high-fidelity mobile prototype for a **Bus Ticket Booking System**.

## Included screens
- **Bus Search & Results**
- **Visual Seat Selection**

## Features
- Clean mobile-first layout
- Design token section for quick visual mapping
- Bus search form with results cards
- Seat legend with color-coded states
- Interactive seat selection with booked, selected, and available states
- Sticky checkout footer with seat summary and fare

## How to run
1. Open `index.html` in any modern browser.
2. Use **Search Buses** to move from the search screen to the seat-selection screen.
3. Click the **back arrow** to return to the first screen.
4. Click available seats to update the selected seats and total fare.
