# Task 1: Kanban Board Simulation

## What is included
- `index.html` is a single, self-contained web page with HTML, CSS, and JavaScript.
- The board has exactly three columns: **To Do**, **In-Progress**, and **Done**.
- Five sample software engineering tasks are included.
- You can move tasks manually with buttons or run the automatic lifecycle simulation.

## How to run
1. Open `index.html` in any modern browser.
2. Click **Run Lifecycle Simulation** to watch the tasks move through the workflow.
3. Use **Reset Board** to return all tasks to **To Do**.

## Folder structure
```text
Task-1/
```

## Uploading to GitHub
1. Keep both files inside the `Task-1` folder.
2. Push the `Task-1` folder to your repository.
3. In your GitHub repository, open `Task-1/index.html` to review the finished lab.
